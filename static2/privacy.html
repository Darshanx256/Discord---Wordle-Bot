Privacy Policy

For Wordle Game Bot

Effective Date: December 9, 2025

This Privacy Policy outlines how the Discord bot Wordle Game (the "Bot") collects, uses, and stores information gathered from its users while providing its Wordle gaming and leaderboard services.

By adding or using the Bot, you consent to the data practices described in this policy.

1. Information We Collect

The Bot is designed to collect only the minimal data necessary to provide its core gaming and ranking features. All data is stored securely in a Supabase database and is never shared with third parties.

1.1 Identifiers (Required for Bot Functionality)

  • User ID (Discord ID): Collected for every user who participates in a game or checks a leaderboard.
    • Purpose: To uniquely identify players, attribute scores/wins, and display usernames on the global/server leaderboards.
  • Guild ID (Server ID): Collected when a game is started or a score is updated.
    • Purpose: To differentiate server-specific leaderboards and to manage the history of used secret words per server (Simple and Classic modes) to ensure word repetition is minimized.

1.2 Content and Game Data (Required for Scoring)

  • Wordle Game Stats (Wins, Total Games): Updated after every completed game in which a user participated.
    • Purpose: To calculate the Bayesian average score and determine player ranking/tier status.
  • Guessed Words & Patterns: Stored temporarily as part of the active game state (e.g., when viewing /wordle_board).
    • Purpose: To display the current game board and keyboard status. This data is deleted immediately upon game completion (win, loss, or stop).
  • Secret Words: Stored in the guild_history and guild_history_classic Supabase tables.
    • Purpose: To track words already used on a server to prevent immediate repetition.
  • In-Memory Message Processing: The Bot temporarily processes message content ONLY when users submit guesses using the -g command during an active game.
    • Purpose: This data is processed in-memory to generate game feedback and is immediately discarded after the game round ends. The Bot does not store, log, or analyze general chat messages.

2. How We Use the Information

The collected data is used exclusively for the operation of the Bot:

  • To display user ranks and scores on the /leaderboard and /leaderboard_global commands.
  • To manage and progress the active Wordle game in a channel.
  • To ensure fairness and replayability by managing the pool of words used in each server.

3. Data Storage and Security

  • Storage Location: All permanent data (User IDs, Guild IDs, Scores) is stored on a secure, external Supabase database.
  • Data Security: Supabase implements industry-standard security measures, including encryption and strict access controls.
  • Data Retention: Game statistics (scores, wins) are stored indefinitely to maintain historical rankings and continuous leaderboards.

4. Data Removal Requests

Users have the right to request the removal of all their personal data from the Bot's database.

To initiate a data removal request:

  1. Send an email to: Ortsx256@proton.me

Upon receiving a confirmed request, we will permanently delete all associated User ID, Guild ID, and scoring data within 30 days.

5. Compliance and Age Limitation

The Bot is not intended for use by children under 13 years of age, which is the minimum age required to use Discord. We do not knowingly collect personal information from children under 13. If you believe we have inadvertently collected such information, please contact us immediately.

If you have any questions or concerns about this policy, please contact Ortsx256@proton.me
